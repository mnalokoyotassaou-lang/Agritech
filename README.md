# Agritech
AgriTech est une application mobile IA qui détecte les maladies des plantes en photo, génère des conseils de traitement en 7 langues africaines. Elle détecte les maladies des plantes en photo, génère des conseils de traitement en 7 langues africaines, avec carte épidémique, alertes météo, chatbot vocal et gestion de parcelles par QR code.
